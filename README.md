# Lexi-Leap Frog Game

A vibrant word puzzle game where players help a frog cross a river by identifying words in specific categories. Features a 4x4 grid, neon aesthetics, and a modern tech stack.

## Features

- **Word Puzzle Gameplay**: Find words in categories to advance
- **4x4 Grid**: Strategic tile selection mechanics
- **Neon Aesthetics**: Bright, engaging visual design
- **Frog Theme**: Help your frog friend cross the river
- **Category-Based**: Words grouped by themes

## Tech Stack

- TypeScript
- React Native (Expo)

## Play

Run with Expo: `npx expo start`
